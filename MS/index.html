# 说说原型

  • 定义：原型是function对象的一个属性，它定义了构造函数制造出的对象的公共祖先，通过该构造函数产生的对象，可以继承该原型的属性和方法，原型也是对象。
    对象中有个隐式的属性**proto，在对象中找属性，如果找不到会通多__proto__属性找到prototype prototype中自带以一个constructor**属性，该属性只想当前构造函数的本身
